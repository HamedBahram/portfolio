## What is NextJS?

NextJS is a React framework for production.

## Why using NextJS?

Using NextJS will eliminate common issues with writing React applications
for production. Client-side routing, etc.

## What is file-based routing?

Export a React component from your `/pages` folder and that will be
rendered as the page.

```jsx
const Home = () => {
  return (
    <>
      <h1>Home Page</h1>
    </>
  )
}

export default Home
```
